JORD is a Single Page Application for eShop, with a minimal of dependencies. Better, Faster, Stronger with Javascript Vanilla.

## Prerequisites
* Javascript Vanilla
* NodeJS
* MongoDB

### NodeJS Dependencies
* Argon2
* MongoDB driver
* Nodemailer
* SASS
* Terser
